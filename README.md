

موقع بورتفوليو شخصي تعرض فيه المشاريع، المعرض، ومعلومات التواصل. تم بناء هذا المشروع باستخدام React وVite بهدف توفير تجربة عرض سريعة وحديثة للمشاريع.

**الميزات**:
- واجهة تفاعلية وسريعة مع تحميل جزئي للمكونات.
- قسم عرض المشاريع (Portfolio) مع صور ووصف لكل مشروع.
- قسم نبذة عني (About) ونموذج تواصل (Contact).
- مؤثرات مرئية خفيفة ومستجيبة عبر Canvas وTailwind CSS.

**التقنيات المستخدمة**:
- React (مبنية عبر Vite)
- Vite (للتطوير السريع وHMR)
- Tailwind CSS (لتصميم الواجهة)
- مكتبات رسومية بسيطة في `src/components/canvas` لعرض مؤثرات ثلاثية الأبعاد

**هيكل المشروع (مختصر)**:
- `src/` : ملفات المصدر الأساسية.
- `src/components/` : مكونات الواجهة (Navbar, Preloader, canvas, sections ...).
- `src/sections/` : مقاطع الصفحة مثل Hero, About, Portfolio, Contact.
- `public/` و `src/assets/` : الأصول (صور، أيقونات، ...).

كيفية تشغيل المشروع محلياً
1. تأكد من تثبيت Node.js (14+ أو أحدث).
2. ثبّت التبعيات:

```bash
npm install
```

3. لتشغيل بيئة التطوير مع HMR:

```bash
npm run dev
```

4. لبناء نسخة الإنتاج:

```bash
npm run build
```

5. لمعاينة نسخة الإنتاج محلياً بعد البناء:

```bash
npm run preview
```

نشر الموقع
- يمكنك نشر المشروع عبر خدمات استضافة ثابتة مثل Vercel أو Netlify بسهولة بربط المستودع وتشغيل الأمر `npm run build` أثناء خطوات النشر.
- أو نشر بسيط على GitHub Pages عبر تهيئة فرع `gh-pages` أو عبر أدوات مثل `gh-pages` إن رغبت.
